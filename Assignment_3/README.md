# Interactive Weather Dashboard

An interactive **Weather Dashboard** built using **HTML5**, **CSS3**, **JavaScript**, and **Chart.js**. The dashboard visualizes weather data for different cities through dynamic charts, allowing users to compare temperature, humidity, and rainfall over a week.

## 📌 Features

- Interactive city selection using a dropdown menu
- Dynamic charts that update instantly based on the selected city
- Displays:
  - 🌡️ Temperature (Line Chart)
  - 💧 Humidity (Bar Chart)
  - 🌧️ Rainfall (Bar Chart)
- Responsive dashboard layout using CSS Grid
- Clean and modern user interface

## 📊 Dashboard Components

- **Temperature Chart** – Displays daily temperatures using a line chart.
- **Humidity Chart** – Shows daily humidity levels using a bar chart.
- **Rainfall Chart** – Visualizes daily rainfall using a bar chart.
- **City Selector** – Switch between Pune, Mumbai, and Delhi to update the charts dynamically.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/interactive-weather-dashboard.git
```

2. Open the project folder.

3. Open **d3.html** in any modern web browser.

No installation is required since **Chart.js** is loaded via CDN.

## 📈 Data Visualization

The dashboard presents weekly weather data for:

- Temperature (°C)
- Humidity (%)
- Rainfall (mm)

Users can switch between cities, and the charts update automatically without refreshing the page.
