# Student Registration Form

A simple and responsive **Student Registration Form** built using **HTML5** and **Bootstrap 5**. This project demonstrates the use of Bootstrap form components to create a clean and user-friendly registration form.

## 📌 Features

- Responsive layout using Bootstrap 5
- Name, Email, and Password input fields
- Gender selection using radio buttons
- Course selection using a dropdown menu
- Submit and Reset buttons
- Clean and modern UI

## 📂 Project Structure

```
│── form.html
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-registration-form.git
```

2. Open the project folder.

3. Open **form.html** in any web browser.

No additional installation is required since Bootstrap is loaded using a CDN.
