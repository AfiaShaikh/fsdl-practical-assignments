# HTML Formatting Tags Table

A simple HTML project that demonstrates various HTML text formatting tags in a clean tabular format. This project is useful for beginners learning HTML and understanding how different text formatting tags work.

## 📌 Features

- Displays commonly used HTML formatting tags
- Includes:
  - Tag name
  - Description
  - Example HTML code
  - Rendered output
- Styled using internal CSS
- Responsive table layout
- Beginner-friendly project

## 📂 Project Structure

```
HTML-Formatting-Tags/
│── index.html
└── README.md
```

## 🏷 HTML Tags Covered

- `<b>` – Bold text
- `<strong>` – Important text
- `<i>` – Italic text
- `<em>` – Emphasized text
- `<mark>` – Highlighted text
- `<small>` – Smaller text
- `<del>` – Deleted text
- `<ins>` – Inserted text
- `<sub>` – Subscript text
- `<sup>` – Superscript text
- `<u>` – Underlined text
- `<code>` – Code formatting
